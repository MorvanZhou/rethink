# Rethink

A note-taking app dependent on python.
The official web version can be found at [https://rethink.run](https://rethink.run).

## Installation

```shell
pip install rethink-note
```

## Usage

```python
import rethink_note

rethink_note.run()
```

```python
import rethink_note

rethink_note.run(
    path='.',
    host='localhost',
    port=8080,
)
```
